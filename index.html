1. Общие сведения

  • Проект: корпоративный сайт для Российско-Германской внешнеторговой палаты
  • Сроки реализации: март 2025 – октябрь 2025 (8 месяцев)
  • Целевые языки: русский, немецкий, английский
  • Текущий сайт: kammer-russland.ru

2. Технические требования

2.1. Технологический стек

Фронтенд

  • HTML5, CSS3, JavaScript
  • Vue.js

Бэкенд

  • FastAPI (Python)
  • Аутентификация: JWT
  • PHP 8.0 (например)

База данных

  • PostgreSQL (основной вариант)
  • Альтернативы: MongoDB, MySQL 8.0

Инфраструктура

  • CI/CD: GitHub Actions (как вариант)

2.2. Функциональные требования

2.2.1. Публичная часть

Главная страница:
  • Основной баннер в виде слайдера (3 слайда)
  • Слайдер актуальных новостей, аналитических обзоров и раздела ВТП рядом
  • Быстрый доступ к основным разделам
  • Языковой переключатель
Новости:
  • Пагинация (12 новостей на страницу)
  • Фильтр по категориям
  • Теги для новостей
  • Кнопка "Поделиться"
Инфографика:
  • Колекция инфографики с фильтрами по дате и темам
Аналитика:
  • Колекция аналитических обзоров с фильтрами по дате и темам
ВТП рядом:
  • Колекция статей с фильтрами по дате и темам
О нас:
  • Коллекция с сотрудниками компании (страница статическая)

2.2.2. Административная панель

Управление контентом:
  • CRUD для всех типов контента
  • Визуальный редактор (TinyMCE 6.x)
  • Загрузка медиа с предпросмотром
  • Планирование публикаций
Пользователи:
  • Ролевая модель (админ, редактор, гость)
  • Журнал действий
  • Двухфакторная аутентификация (под вопросом)
SEO-инструменты:
  • Генератор метатегов
  • Управление sitemap.xml
  • Валидатор микроразметки

2.3. Дизайн-требования

2.3.1. Визуальный стиль

  • Цветовая схема: Корпоративные цвета (дополняется)
  • Типографика: Современная и читаемая (дополняется)

2.3.2. UI-компоненты

  • Модульная сетка (12 колонок)
  • Карточки с hover-эффектами
  • Progress bar для загрузки
