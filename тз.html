Техническое задание на разработку корпоративного сайта

1. Общие сведения

Проект: Корпоративный сайт для Kammer Russland

Сроки реализации: Март 2025 – Октябрь 2025 (8 месяцев)

Целевые языки: Русский, немецкий, английский

Текущий сайт: kammer-russland.ru

2. Технические требования

2.1. Технологический стек

  • Фронтенд: HTML5, CSS3, JavaScript (ES6+), Vue.js (версия 3.x), Axios, Webpack/Vite
  • Бэкенд: FastAPI (Python 3.9+), JWT аутентификация
  • База данных: PostgreSQL 14+ (основной вариант), MongoDB 5.0+, MySQL 8.0
  • Инфраструктура: Docker, Nginx, CI/CD через GitHub Actions

2.2. Функциональные требования

2.2.1. Публичная часть

  • Главная страница: слайдер новостей, блок "Цифра дня", быстрый доступ к разделам, языковой переключатель.
  • Новости: пагинация (12 новостей на страницу), фильтры, теги, кнопка "Поделиться".
  • Инфографика: интерактивные графики (Chart.js), скачивание в PNG/PDF.
  • Аналитика: категории статей, поиск, рекомендуемые материалы.
  • О нас: интерактивная карта, галерея команды, история организации (таймлайн).

2.2.2. Административная панель

  • Управление контентом: CRUD, TinyMCE 6.x, загрузка медиа, планирование публикаций.
  • Пользователи: ролевая модель, журнал действий, двухфакторная аутентификация.
  • SEO-инструменты: генератор метатегов, управление sitemap.xml, валидатор микроразметки.

2.3. Дизайн-требования

  • Цветовая схема: основные цвета — синий (#3498db), белый (#fff), серый (#f9f9f9).
  • UI-компоненты: модульная сетка, карточки с hover-эффектами, аккордеоны, кастомные элементы формы.

2.4. SEO и производительность

  • Скорость загрузки: ≤2s (Desktop), ≤3s (Mobile).
  • Lighthouse score ≥90.
  • Lazy loading для изображений, оптимизация WebP, Schema.org разметка.
